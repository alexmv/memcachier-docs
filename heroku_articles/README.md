# Heroku Articles on Caching / MemCachier

~~~~
devcenter pull https://devcenter.heroku.com/articles/memcachier
vim memcachier.md
devcenter push memcachier.md
~~~~

The `devcenter` CLI can be found here: https://github.com/heroku/devcenter-cli
